Personal Experience Database with Bucket List Manifestation

Mobile app for tracking life experiences and achieving dreams. Offline-first with SQLite storage, built with Laravel + Livewire + NativePHP.

Get Started Learn More

Core Features

Everything you need to capture life's moments and achieve your dreams.

📸

Events Management

Full CRUD operations for events with photos. Categories include Food & Dining, Music, Travel, Activities, Culture, and more. 1-5 star rating system with search and filter functionality.

🎯

Dreams (Bucket List)

Full CRUD operations for dreams and goals. Priority system with 1-5 stars. Track achievements with celebrations, set target dates, and link achieved dreams to events.

📊

Dashboard & Statistics

Real-time metrics for events and dreams. View recent activities, achievements, completion rates, and category breakdowns for both events and dreams.

📱

Mobile Features

Bottom navigation: Home, Events, Dreams, Stats. Offline-first with SQLite storage. Touch-optimized UI with native photo capture and gallery integration.
